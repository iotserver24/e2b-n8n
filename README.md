# xibe-n8n

E2B sandbox builder & launcher for n8n with auto-resume.

## Repository Structure

This repository has been redesigned for better organization:

- [**`n8n/`**](file:///home/r3ap3reditz/codes/xibe-n8n/n8n): The core TypeScript codebase for building and running n8n templates.
- [**`docs/`**](file:///home/r3ap3reditz/codes/xibe-n8n/docs): Project documentation, guides, and images.
- [**`e2b-sandboxes/`**](file:///home/r3ap3reditz/codes/xibe-n8n/e2b-sandboxes): A collection of individual E2B sandbox configurations.

## Getting Started

To work with the n8n automation logic, navigate to the `n8n` directory:

```bash
cd n8n
npm install
npm run build:template # Build the n8n template
npm start              # Start the n8n sandbox
```

For more detailed information, please refer to the [documentation](file:///home/r3ap3reditz/codes/xibe-n8n/docs/README.md).
